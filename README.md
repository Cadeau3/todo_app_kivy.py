# Productivity App (Kivy & Python)

A simple productivity application built in **Python** using **Kivy** (GUI) for mobile and desktop platforms.  
Tracks tasks with categories, priorities, due dates, and completion status.

## Features

- Add, view, complete, and delete tasks
- Tasks categorized by School, Work, Personal, or General
- Due date and priority management
- Export tasks to CSV
- Scrollable, mobile-friendly GUI
- Persistent storage using `tasks.json`

## Skills Demonstrated

- Python programming (data structures, file handling, JSON, CSV)
- GUI development using Kivy
- User interface design for usability
- Debugging and testing
- Data persistence and export
- Version control (GitHub)
